# ロゴアセット

AI.LandBase のロゴファイルを管理するディレクトリ。
アプリケーションからロゴを参照する場合は `public/logo/` 配下のファイルを使用すること。

## 正規ファイル

- `logo.png` — 1024x1024 RGBA PNG（Drive 原本と同一）
- 必要に応じてこのファイルからリサイズ・形式変換して派生版を生成する

## 原本

Google Drive: https://drive.google.com/uc?export=view&id=11zxS-CXtXcRDu8lznBwdCdqGaZVPSqho

## 命名規約

| ファイル名 | 用途 | 背景色 | 解像度 | 形式 | ステータス |
|-----------|------|--------|--------|------|-----------|
| `logo.png` | 基本ロゴ | 透過 | 1024x1024 | PNG | 配置済み |
| `logo.svg` | スケーラブル用 | 透過 | - | SVG | 未作成（要素材入手） |
| `logo-mark.png` | シンボルのみ（テキストなし版） | 透過 | TBD | PNG | 未作成 |
| `logo-white.png` | 暗背景用（白抜き） | 暗色向け | TBD | PNG | 未作成 |
| `favicon.ico` | ブラウザタブ | 透過 | 32x32 | ICO | 未作成 |
| `og-image.png` | OGP 用 | 要設計 | 1200x630 | PNG | 未作成 |
